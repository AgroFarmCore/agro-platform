# Module Ownership

Рекомендуемый стартовый ownership:

- `packages/database`: tech lead + 1 backend owner
- `packages/modules/access`: tech lead
- `packages/modules/order`: backend owner
- `packages/modules/recommendation`: backend owner
- `apps/web`: frontend/product owner
- `apps/portal`: shared frontend owner
- `apps/backoffice`: operations owner
- `apps/worker`: backend owner

CODEOWNERS должен отражать эти же зоны ответственности.
